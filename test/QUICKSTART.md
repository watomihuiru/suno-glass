# Быстрый старт с тестами

## Установка

Зависимости уже установлены. Если нужно переустановить:

```bash
npm install
```

## Запуск тестов

### Все тесты сразу
```bash
npm test
```

### Только бэкенд
```bash
npm run test:backend
```

### Только фронтенд
```bash
npm run test:frontend
```

### С покрытием кода
```bash
npm run test:coverage
```

## Что тестируется

### Бэкенд
- ✅ ApiRequestService - работа с API запросами
- ✅ SocketHandlers - обработка Socket.IO событий
- ✅ PollingService - опрос статуса задач
- ✅ Server - базовые функции сервера

### Фронтенд
- ✅ ValidationUtils - валидация форм
- ✅ Utils - вспомогательные функции

## Структура

```
test/
├── backend/          # Тесты серверной части
└── frontend/         # Тесты клиентской части
```

## Примечания

- Тесты используют моки для внешних зависимостей
- Не требуют реального API ключа
- Не требуют запущенного сервера (кроме некоторых интеграционных тестов)
